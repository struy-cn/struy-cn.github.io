AI对话大模型哪家强？

没有对比的话可能看不出来，现在的对话大模型几乎都存在胡编乱造的问题，还有就是计算和逻辑基本上都有所欠缺，不是那么准确。

由于受限的条件，没有感受过GPT4的魔力，但是其他能体验的模型基本上也都玩了一遍；

这还得感谢 ChatAll🔗 这个项目把市面上几乎所有的大模型都放到一起，我才有机会让他们PK，看看哪个更准确。

准确性来说肯定是 Bing 结合在线搜索会准确一些，但是那晚给团队分享时一个乘法运算 Bing 给我算错了，有点出乎意料，话说它后面真的由 GPT4 驱动吗。所以大家问的时候悠着点。

值得关注的是：Claude 已经成为 OpenAI 的强劲对手，从问答体验上来说不输 ChatGPT，并且据网上各位大佬的体验反馈，其多轮对话能力很强，经常需要手动清除会话，不然上下文他记得太清楚了，但此时你的上下文可能已经变了。

据了解，由AI写作再次出圈的 Notion AI 貌似也是由 Claude 驱动。

AI 绘图真的能替代原画师吗？

这个问题我不是专业的，所以我也不知哈哈哈。
最近看到一个新闻说，有个人用AI绘画出了一本漫画，但是为了保持风格一致费了不少劲；而且有些生成的图不能直接使用，需要二次加工。但是这或许会真的改变了很多内容的创作方式。

以后看到漫画和动画怕是很多由AI生成了；之前用 SD 也生成了一些图，对于不是设计师的人来说，满足了那种创造欲，而且 SD 插件丰富，可玩性很高。

可以参考我之前的SD文章，一起体验下: 本地白嫖AI绘画 ，Stable Diffusion 初探！

AI写作，拓宽思路

AI 写作和 ChatGPT 对话一样，有时候你能看到很多正确的废话和啰嗦的总结，但是有时候的确是可以给你提供一些想法和思路，比如有些事情你就是找不到更好的词来描述，也许他知道怎么去描述。

AI 写作这样的场景，我个人觉得是对接大模型最简单的场景，因为只需要预设置好角色提示语和问题的摘要，总能给出一系列答案，这些答案也总有能有一部分击中你需求，实在不行就让 AI 多写几遍嘛。

文字、文档、知识库

除了AI写作，日常的工作、生活、学习，打交道最多的就是文字、文档，以及各种资料书籍。自从 ChatGPT 火爆之后，围绕其探索产品就层出不穷

比如 ChatDoc、ChatPDF 等等，基于文档问答和信息汇总的工具，很受研究、学习的人欢迎

B站视频地址

文档对话和分析的 ChatDOC 和 ChatPDF 实现原理也大同小异（这类应用基本上都离不开：llama-index 和 langchain，但我不是这方面的专业人士哈哈，就不误导了），就看谁体验更好，更准确。

以及智能生成 PPT，比如 MindShow，很多人通过 GPT 生成 PPT大纲再让 MindShow 渲染，省去排版和美化的麻烦。挺方便

私有知识库

还有一些场景是企业需要接入这类大模型，做私有知识库的文档或者智能客服，但是企业的隐私数据又不想发送给这些大模型平台，这样的数据安全问题也是值得考虑的。

那怎么做呢，实际上我也在找，对于一些商用大模型，一些人在观望，一些人已经达成合作；

还有一些在做开源尝试。

AI编程，效率至上

AI编程不用说了，作为程序员，这个效率工具高低得整一个吧。片头提到的书签合集里包含了编码相关的几个插件，几乎都试用过了哈，编程效率的确提升了不少，主要是一些体力活和常见的逻辑他能及时给你提示，免去你费力敲一堆的代码。

我现在主力使用的 ChatGPT 生成完整代码，主要是独立的脚本这类 + Github Copilot 基于注释自动补全一些逻辑代码，GPT API 快到期了，后续准备使用 Claude、Bing 继续白嫖。

AI影音，数字人卖货到底有没有人买？
