Github 高星

ILLA Cloud 是一个开源低代码开发者工具，去年 9 月在 Github 上线以来已经收获了 6千多个 Star。

一拖一拉快速开发

其主要亮点是拖拉拽的方式通过组件库构建页面布局，并且支持多页面布局，其有丰富的UI组件，比如表格，图表，地图等，绝对能满足你的各类业务需求。

多样数据源

还可以通过页面操作的方式来快速链接多样的数据源，比如 PostgreSQL、MySQL、ClickHouse、Elastic Search 等常见数据源。

智能 SQL

在页面可以通过 AI 结合你提供的 语言描述 生成需要的 SQL 语句，提高数据开发效率。

Hugging Face API 支持

值得一提的是：工具支持 Hugging Face API 的接入，可以快速调用多种大模型，从而实现有AI能力的工具快速开发。

灵活部署

ILLA Cloud 还有领先的协作功能，用户可以在这里通过协作方式快速构建页面布局；另外它还提供了丰富的页面模板和专业的文档。支持 云端体验和私有化部署
