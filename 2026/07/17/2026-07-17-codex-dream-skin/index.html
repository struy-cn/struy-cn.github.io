你用 AI 编程工具的时候，有没有想过一个问题：这界面能不能换个皮肤？

VS Code 能换主题，IDEA 能换主题，Cursor 也能换——凭什么 Codex 桌面端就只能用默认脸？

有人动手了。

🎨 Codex Dream Skin：给 Codex 换一张会呼吸的脸

GitHub 上出现了一个项目叫 Codex Dream Skin，专干一件事：给 OpenAI Codex 桌面端换肤。

不是那种改配置文件换个配色方案的换肤——是真·换背景图、换氛围、换心情的那种。粉系定制、财神打工版、红白科幻、紫夜限定、舞台黑金……八种风格任你选。

上面两张是真实的 Codex 桌面端注入效果——侧栏、卡片、项目选择、输入框全是原生控件，只是在底下铺了一层主题背景。

而且它做了一件很硬核的事：不改官方安装包，不碰代码签名，用 CDP（Chrome DevTools Protocol）注入实现换肤。

🔧 技术方案：CDP 注入，举重若轻

Codex 桌面端底层是 Electron（ Chromium 内核），这意味着它天然支持 Chrome DevTools Protocol。这个项目的核心思路非常优雅：

  1. 启动 Codex 时通过本机回环（127.0.0.1）挂载 CDP
  2. 注入自定义 CSS + 装饰 DOM 节点
  3. 在原生控件下层铺一张 16:9 背景图

就这么简单。侧栏、建议卡、项目选择器、输入框——全是 Codex 原生控件，照常点击照常输入。只是在它们下面，多了一层你选的背景。

关键安全设计：

  • CDP 只绑 127.0.0.1，不暴露到网络
  • 不修改 .app / app.asar / WindowsApps 目录
  • 不触碰代码签名
  • 不静默改写 API Key / Base URL（换肤和中转配置完全隔离）
  • 一键可恢复官方外观

这不是 hack，是 Electron 应用的标准调试协议的创意用法。

🖼️ 八种预设风格

项目内置了八种视觉方向的概览效果图：

  • 粉系定制 — 樱花、花瓣、柔光
  • 财神打工版 — 金色配色，打工人的精神图腾
  • 红白科幻 — 红白配色，科技感十足
  • 清透定制 — 简洁通透
  • 灵感小宇宙 — 紫色星空
  • 紫夜限定 — 暗紫色系
  • 青蓝虚拟歌姬 — 二次元风格
  • 舞台黑金 — 黑金奢华

macOS 版还预置了多个抽象主题包（midnight-aurora、sakura-dawn、amber-dusk、forest-mist、cyber-neon），不用人物也能很美。

📝 最惊艳的：提示词工程文档

这个项目最让我意外的不是换肤功能本身，而是它附带了一份极其详细的背景图生成提示词指南。

这不是随便写写”生成一个粉色背景”就完事了。这份文档定义了：

精确构图规范：

  • 主视觉中心放在 x=68%~76%
  • 脸部和手部控制在 x=62%~88%
  • 左侧 x=0%~52% 为”文字安全区”，低对比度、低信息密度
  • 所有关键内容距边缘至少 8%

浅色/暗色兼容要求：

  • 安全区要有连续的低频明暗变化
  • 避免纯白烧穿或纯黑死区
  • 同一张图在浅色壳上能承载深色文字，暗色壳叠加遮罩后仍保留层次

输入图职责分离：

  • Image 1 = UI 效果参考（只看色板、光线、构图）
  • Image 2 = 纯风格/场景参考（材质、花材、调色）
  • Image 3 = 可选的身份参考（仅在确认授权后使用）

甚至还提供了一份完整的负面提示词：

screenshot, UI, software window, sidebar, panel, card, rounded rectangle, button, icon, input box, composer, code editor, cursor, device frame, poster layout, typography, letters, words, watermark, split screen, collage, polaroid, blank white panel, duplicate person, extra limbs, malformed hands, cropped face, cropped hands

这个精细程度，已经超过大多数 AI 绘画教程。这哥们是把给 Codex 换肤当论文写了。

🤔 我的评价

亮点：

  • 技术方案选得好。 CDP 注入是 Electron 应用换肤的最优解——不改二进制、不碰签名、可恢复。比那些修改 app.asar 的方案安全得多
  • 文档质量极高。 提示词工程指南本身就有独立价值，哪怕你不用这个换肤工具，也能学到不少 AI 绘画的构图知识
  • 双平台支持。 macOS 和 Windows 都有完整脚本
  • 安全意识在线。 明确声明非官方产品、CDP 只绑本机回环、换肤和中转配置隔离

不足：

  • 依赖 Codex 桌面端，OpenAI 一旦更新 UI 结构可能就挂
  • 预设主题偏二次元/人物向，抽象/极简选项可以再丰富些
  • README 顶部的赞助商（passion8.cc AI 中转服务）植入有点生硬

适合谁？

  • 天天用 Codex 桌面端写代码的人
  • 喜欢个性化工作环境的开发者
  • 对 Electron 逆向 / CDP 注入感兴趣的技术好奇者
  • 想学 AI 绘画提示词工程的人（光看文档就值了）
